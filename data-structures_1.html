Introduction to Data Structures

Data structures are fundamental concepts in programming, providing ways to organize, manage, and store data efficiently. They enable developers to perform operations like searching, sorting, and modifying data with ease. This section introduces two basic yet powerful data structures in Python: strings and lists.

By mastering these data structures, you'll gain the tools to handle real-world problems, manipulate datasets, and build robust applications.

Python Strings

Strings are sequences of characters, commonly used for text manipulation. They are immutable, meaning their content cannot be changed after creation.

Creating Strings

Strings can be created using single or double quotes:

greeting = 'Hello'
name = "Alice"

Basic String Functions

  • len(): Returns the length of the string.
  • in: Checks if a substring exists within a string.
text = "Python Programming"
print(len(text))  # Output: 18
print("Python" in text)  # Output: True
print("Java" in text)    # Output: False

String Indexing and Slicing

Strings are indexed using zero-based indexing, and slicing allows for extracting substrings. Slicing is done using the syntax [start:stop:step], where:

  • start: The index where the slice begins (inclusive). If omitted, it defaults to the beginning of the string (index 0).
  • stop: The index where the slice ends (exclusive). If omitted, it defaults to the end of the string.
  • step: The interval between characters in the slice. If omitted, it defaults to 1.
text = "Hello, World!"

# Examples:
print(text[0])        # Output: "H" (first character)
print(text[-1])       # Output: "!" (last character)

# Slicing with start and stop:
print(text[7:12])     # Output: "World" (slice from index 7 to 11)
print(text[:5])       # Output: "Hello" (first 5 characters)
print(text[7:])       # Output: "World!" (from index 7 to the end)

# Slicing with step:
print(text[::2])      # Output: "Hlo ol!" (every second character)
print(text[::-1])     # Output: "!dlroW ,olleH" (reversed string)

# Combining start, stop, and step:
print(text[2:9:2])    # Output: "lo " (characters at indices 2, 4, 6, 8)
print(text[1:12:3])   # Output: "e,Wr" (characters at indices 1, 4, 7, 10)

Using slicing, you can manipulate and access parts of strings flexibly, making it a powerful tool for string operations.

Looping Through Strings

You can iterate over a string using a for loop:

text = "Python"
for char in text:
    print(char)
# Output:
# P
# y
# t
# h
# o
# n

Modifying Strings

While strings are immutable, you can create modified versions using methods like upper(), lower(), and strip():

text = "  Hello, World!  "
print(text.upper())  # "  HELLO, WORLD!  "
print(text.lower())  # "  hello, world!  "
print(text.strip())  # "Hello, World!" (removes leading and trailing spaces)

String Concatenation

Combine strings using the + operator:

greeting = "Hello"
name = "Alice"
message = greeting + ", " + name + "!"
print(message)  # Output: "Hello, Alice!"

String Formatting

Python provides multiple ways to format strings:

  • Using f-strings (introduced in Python 3.6):
    • name = "Alice"
age = 25
print(f"My name is {name} and I am {age} years old.")
# Output: "My name is Alice and I am 25 years old."
  • Using the format() method:
    • name = "Alice"
age = 25
print("My name is {} and I am {} years old.".format(name, age))
# Output: "My name is Alice and I am 25 years old."

Common String Methods

  • lower(): Converts to lowercase.
  • upper(): Converts to uppercase.
  • strip(): Removes leading and trailing spaces.
  • replace(): Replaces substrings.
  • split(): Splits into a list.
  • join(): Joins a list into a string.
  • find(): Returns the index of the first occurrence of a substring.
  • startswith(): Checks if the string starts with a specified prefix.
  • endswith(): Checks if the string ends with a specified suffix.
text = "  Hello, World!  "
print(text.strip())              # Removes whitespace
print(text.replace("World", "Python"))
print(text.find("World"))        # Returns index 8
print(text.startswith("Hello"))  # Output: False (leading spaces)
print(text.endswith("!  "))      # Output: True

Python Lists

Lists are versatile, mutable collections that can hold elements of different types. They are one of the most commonly used data structures in Python.

Creating Lists

Lists are defined using square brackets []:

numbers = [1, 2, 3]
fruits = ["apple", "banana", "cherry"]
mixed = [1, "apple", 3.14, True]

Accessing Items

Items in a list can be accessed using **zero-based indexing** for normal indexing and **negative indexing** to count from the end of the list. Slicing allows you to retrieve a portion of a list by specifying a start, stop, and optional step value. 

fruits = ["apple", "banana", "cherry", "date", "elderberry"]

# Normal indexing:
print(fruits[0])  # Output: "apple" (first item)
print(fruits[2])  # Output: "cherry" (third item)

# Negative indexing:
print(fruits[-1])  # Output: "elderberry" (last item)
print(fruits[-2])  # Output: "date" (second-to-last item)

# Slicing (start:stop):
print(fruits[1:4])  # Output: ["banana", "cherry", "date"] (items from index 1 to 3)

# Slicing (with omitted start or stop):
print(fruits[:3])   # Output: ["apple", "banana", "cherry"] (first three items)
print(fruits[2:])   # Output: ["cherry", "date", "elderberry"] (from index 2 to the end)

# Slicing with step (start:stop:step):
print(fruits[::2])  # Output: ["apple", "cherry", "elderberry"] (every second item)
print(fruits[1::2]) # Output: ["banana", "date"] (every second item starting from index 1)

# Reversing a list with slicing:
print(fruits[::-1]) # Output: ["elderberry", "date", "cherry", "banana", "apple"]

Slicing creates a new list and does not modify the original list. The start index is inclusive, while the stop index is exclusive. When using step, it specifies the interval between elements to include in the slice (default is 1).

Checking if an Item Exists

Use the in keyword to check if an item exists in a list:

fruits = ["apple", "banana", "cherry"]

print("banana" in fruits)  # Output: True
print("orange" in fruits)  # Output: False

Changing Item Values

Lists are mutable, so you can change the value of items by assigning a new value to a specific index:

fruits = ["apple", "banana", "cherry"]
fruits[1] = "blueberry"
print(fruits)  # Output: ["apple", "blueberry", "cherry"]

Adding Items

Add items to a list using append() to add to the end, or insert() to add at a specific index:

fruits = ["apple", "banana"]

# Add to the end:
fruits.append("cherry")
print(fruits)  # Output: ["apple", "banana", "cherry"]

# Add at a specific index:
fruits.insert(1, "blueberry")
print(fruits)  # Output: ["apple", "blueberry", "banana", "cherry"]

Removing Items

Remove items using remove() (by value), pop() (by index), or del (by index or slice):

fruits = ["apple", "banana", "cherry"]

# Remove by value:
fruits.remove("banana")
print(fruits)  # Output: ["apple", "cherry"]

# Remove by index:
fruits.pop(0)
print(fruits)  # Output: ["cherry"]

Looping Through Lists

Loop through a list using a for loop to access and process each element:

fruits = ["apple", "banana", "cherry"]

# Print each item:
for fruit in fruits:
    print(fruit)

# Output:
# apple
# banana
# cherry

List Comprehension

List comprehensions provide a concise way to create lists. They allow you to generate a new list by applying an expression to each item in an existing list or iterable. 

# Generate a list of squares:
numbers = [1, 2, 3, 4, 5]
squares = [x**2 for x in numbers]
print(squares)  # Output: [1, 4, 9, 16, 25]

# Filter a list:
fruits = ["apple", "banana", "cherry", "kiwi"]
short_fruits = [fruit for fruit in fruits if len(fruit) <= 5]
print(short_fruits)  # Output: ["apple", "kiwi"]

Practice Questions

1. Count Words in a Sentence

Write a program that counts and prints the number of words in a given sentence.

Solution 
sentence = "Learning Python is fun and rewarding"
words = sentence.split()
print(len(words))

2. Find the Longest Word

Write a program that finds and prints the longest word in a list of words.

Solution 
words = ["apple", "banana", "cherry", "dragonfruit", "grape"]
longest_word = max(words, key=len)
print(longest_word)

3. Filter Words Starting with a Specific Letter

Write a program that filters and prints words from a list that start with the letter "A".

Solution 
words = ["apple", "banana", "avocado", "grape", "apricot"]
filtered_words = [word for word in words if word.lower().startswith('a')]
print(filtered_words)

4. Reverse Each Word

Write a program that reverses and prints each word in a list of strings.

Solution 
words = ["hello", "world", "python", "rocks"]
reversed_words = [word[::-1] for word in words]
print(reversed_words)

5. Concatenate Strings in a List

Write a program that concatenates and prints all strings in a list into a single string separated by spaces.

Solution 
words = ["Python", "is", "a", "powerful", "language"]
result = " ".join(words)
print(result)

6. Find Words Containing a Specific Substring

Write a program that finds and prints all words in a list that contain the substring "an".

Solution 
words = ["banana", "apple", "orange", "mango", "grape"]
substring = "an"
filtered_words = [word for word in words if substring in word]
print(filtered_words)

7. Convert Strings to Uppercase

Write a program that converts and prints all strings in a list to uppercase.

Solution 
words = ["hello", "world", "python", "rocks"]
uppercase_words = [word.upper() for word in words]
print(uppercase_words)

8. Sort Strings Alphabetically

Write a program that sorts and prints a list of strings in alphabetical order.

Solution 
words = ["banana", "apple", "cherry", "date", "fig"]
sorted_words = sorted(words)
print(sorted_words)
