Overview

This week, we will explore Python's versatile data structures, focusing on lists and strings. These are foundational tools for organizing, processing, and manipulating data.

Python Lists

A list is an ordered, mutable collection of elements. Lists can store elements of any data type and provide a wide range of operations for manipulating their contents.

Creating Lists

Lists are created using square brackets []:

# Creating a list
numbers = [1, 2, 3, 4]
fruits = ["apple", "banana", "cherry"]
mixed = [1, "hello", 3.5, True]

Accessing Elements

Elements in a list are accessed by their index. Python uses zero-based indexing: 

fruits = ["apple", "banana", "cherry"]
print(fruits[0])  # Output: "apple"
print(fruits[-1])  # Output: "cherry" (negative indexing starts from the end)

Modifying Lists

Lists are mutable, so their elements can be changed:

fruits[1] = "blueberry"
print(fruits)  # Output: ["apple", "blueberry", "cherry"]

Common List Methods

  • append(): Adds an element to the end of the list.
  • insert(): Inserts an element at a specific position.
  • remove(): Removes the first occurrence of a value.
  • pop(): Removes and returns an element by index.
  • sort(): Sorts the list in ascending order.
  • reverse(): Reverses the list order.
fruits.append("orange")
fruits.insert(1, "grape")
fruits.remove("banana")
popped = fruits.pop(0)  # Removes "apple"
fruits.sort()  # Alphabetical order
fruits.reverse()  # Reversed order

Python Strings

Strings are sequences of characters, used to store and manipulate text. Strings are immutable, meaning their content cannot be changed after creation.

Creating Strings

Strings can be created using single or double quotes:

greeting = 'Hello'
name = "Alice"

String Operations

Strings support a variety of operations, such as concatenation, repetition, and slicing: 

greeting = "Hello"
name = "Alice"
message = greeting + ", " + name  # Concatenation
print(message)  # Output: "Hello, Alice"

repeated = greeting * 3
print(repeated)  # Output: "HelloHelloHello"

String Methods

Python provides many built-in methods for strings:

  • lower(): Converts the string to lowercase.
  • upper(): Converts the string to uppercase.
  • strip(): Removes leading and trailing whitespace.
  • replace(): Replaces occurrences of a substring with another.
  • split(): Splits the string into a list based on a delimiter.
  • join(): Joins elements of a list into a single string using a delimiter. 
text = "  Hello, World!  "
print(text.lower())  # "  hello, world!  "
print(text.upper())  # "  HELLO, WORLD!  "
print(text.strip())  # "Hello, World!"
print(text.replace("World", "Python"))  # "  Hello, Python!  "

words = text.split(", ")
print(words)  # ["  Hello", "World!"]
joined = " - ".join(words)
print(joined)  # "  Hello - World!"

String Slicing

Strings can be sliced similarly to lists:

text = "Hello, World!"
print(text[0:5])  # "Hello"
print(text[-6:])  # "World!"
print(text[::-1])  # "!dlroW ,olleH" (reversed)
