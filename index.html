Overview

This week, we will focus on the basics of programming, the foundational rules of Python, and core concepts like variables, arithmetic operations, and logical operators.

Programming Basics

Programming involves writing instructions that a computer can execute. These instructions are written in a programming language, such as Python.

Core concepts include:

  • Syntax: The set of rules that define the structure of a programming language.
  • Variables: Containers for storing data values.
  • Data Types: The classification of data items, such as integers, floats, strings, and booleans.

Python Basics & Rules

Python is known for its simple syntax and readability. Here are some key rules to follow when writing Python code:

  • Python code is case-sensitive.
  • Use indentation to define code blocks (e.g., for loops or conditionals).
  • Comments in Python start with a `#` symbol and are not executed.
  • Python files typically have a `.py` extension.

Example of a simple Python code:

# This is a comment
name = "Alice"
print("Hello, " + name)

Variables

Variables are like containers for storing data values. You can use them to hold numbers, text, and other types of data.

Example of variable declaration and usage:

myName = "Louis"
print("I am " + myName)

Here, `myAge` is a variable that stores the number `22`. The `print` function is used to display the value of `myAge` along with some text.

Python Data Types

Understanding data types is crucial for writing efficient code. Here are the primary data types in Python:

  • Integer: Whole numbers, e.g., `42`
  • Float: Decimal numbers, e.g., `3.14`
  • String: Text data, e.g., `"Hello, World!"`
  • Boolean: True or False values, e.g., `True`
  • List: Ordered sequences of items, e.g., `[1, 2, 3]`
  • Dictionary: Key-value pairs, e.g., `{"name": "Alice", "age": 25}`

Examples:

# Integer
age = 22

# Float
height = 5.9

# String
greeting = "Hello, World!"

# Boolean
is_student = True

# List
fruits = ["apple", "banana", "cherry"]

# Dictionary
person = {"name": "Alice", "age": 25}

Arithmetic Operators

Arithmetic operations allow you to perform calculations using numbers. Python supports basic arithmetic operators:

  • Addition: `+`
  • Subtraction: `-`
  • Multiplication: `*`
  • Division: `/`
  • Floor Division: `//`
  • Exponentiation: `**`
  • Modulus: `%`

Examples:

# Addition
sum = 5 + 3
print(sum)
# Subtraction
difference = 10 - 4
print(difference)
# Multiplication
product = 7 * 6
print(product)
# Division
quotient = 20 / 4
print(quotient)
# Floor Division
floored_result = 15 // 2
print(floored_result)
# Exponentiation
power = 2 ** 3
print(power)
# Modulus: Remainder after Division
remainder = 10 % 3
print(remainder)

Logical (Comparison) Operators

Comparison operators allow you to compare two values and return a Boolean result (`True` or `False`). They are essential in decision-making within your code.

  • Equal To: `==`
  • Not Equal To: `!=`
  • Greater Than: `>`
  • Less Than: `<`
  • Greater Than or Equal To: `>=`
  • Less Than or Equal To: `<=`

Examples:

# Equal To
result = 5 == 5  # True

# Not Equal To
result = 5 != 5  # False

# Greater Than
result = 7 > 5  # True

# Less Than
result = 5 < 6  # True

# Greater Than or Equal To
result = 5 >= 5  # True

# Less Than or Equal To
result = 7 <= 5  # False

print(result)

Reading Input from Users

In Python, you can use the input() function to read input from users. This function displays a prompt and waits for the user to enter some text. The input is returned as a string.

Basic Usage

To read input from the user, use the input() function and optionally provide a prompt message. Here is a basic example: 

user_name = input("Enter your name: ")
print("Hello, " + user_name + "!")

Reading Numeric Input

If you need to read numeric input, you'll need to convert the input from a string to a number using int() for integers or float() for floating-point numbers. Here is an example: 

age = int(input("Enter your age: "))
print("You are " + str(age) + " years old.")

b = -3

print(discriminant)

5. BMI Calculator

height = 1.75
bmi = weight / (height ** 2)
print(bmi)
