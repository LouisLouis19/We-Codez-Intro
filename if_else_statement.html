Overview

This section introduces conditional statements in Python, with a focus on the if/else structure. Conditional statements are essential for decision-making in programming, allowing your code to perform different actions based on specific conditions. By understanding and using if and else statements effectively, you can make your programs more dynamic and responsive.

If/Else Statements

Conditional statements enable your program to make decisions and execute certain blocks of code depending on whether specific conditions are met. The if/else structure is one of the most common ways to create conditional logic, making it a fundamental tool for controlling program flow.

Basic Structure

The syntax for an if/else statement in Python is simple: 

if condition:
    # Code to execute if the condition is True
else:
    # Code to execute if the condition is False
  • If: The if keyword introduces the condition. If the condition evaluates to True, the code inside the if block executes.
  • Else: The else block is optional and only runs if the if condition is False. If there’s no else block, the program skips it and moves on.
  • Condition: The condition is typically an expression that evaluates to a Boolean value (True or False). Python will evaluate the condition, and only one of the blocks will execute based on whether the condition is met.

Example

Let’s look at a simple example to demonstrate the if/else structure:

age = 20

if age >= 18:
    print("You are an adult.")
else:
    print("You are a minor.")

In this example, the program checks if the variable age is 18 or older. If the condition is True, it prints "You are an adult." If it’s False, it prints "You are a minor."

Nested If/Else Statements

You can nest if/else statements within each other to handle more complex conditions. This structure is especially helpful when decisions depend on multiple related factors. 

age = 17
has_permission = True

if age >= 18:
    print("You are an adult.")
else:
    if has_permission:
        print("You are a minor, but you have permission.")
    else:
        print("You are a minor and do not have permission.")

In this example, if age is under 18, a second if statement checks whether has_permission is True. This nested approach allows for more detailed decision-making.

Common Pitfalls

  • Indentation Errors: Each block of code inside if, elif, or else statements must be consistently indented. In Python, indentation is crucial for defining the scope of these blocks.
  • Order of Conditions: In a series of if and elif statements, Python stops evaluating once it finds a True condition. Make sure conditions are ordered logically to avoid unintended results.

Elif and Else Statements

The if statement allows us to execute code based on a condition. However, when you need to check multiple conditions, using just an if statement can be inefficient. This is where elif (short for "else if") and else come into play.

Elif statements help handle multiple conditions. Once a condition in an if or elif block evaluates as true, the corresponding block of code is executed, and the remaining conditions are ignored.

The else block serves as a "catch-all" for situations where none of the if or elif conditions are met. It ensures that your code still responds to inputs even when they don't match any specific condition.

Example:

score = 85
        
if score >= 90:
    print("Grade: A")
elif score >= 80:
    print("Grade: B")
elif score >= 70:
    print("Grade: C")
else:
    print("Grade: F")

Explanation: In this example, the program checks a score to determine the appropriate grade. The first condition checks if the score is greater than or equal to 90. If true, it prints "Grade: A". If false, it moves to the next elif statement, checking if the score is greater than or equal to 80, and so on. Finally, if none of the conditions are true, the else block runs, printing "Grade: F". Note that once a condition is true, the remaining conditions are skipped.

This conditional structure ensures flexibility in decision-making by providing multiple options for varying conditions. It also keeps your code clean and easy to follow, avoiding complex and nested if statements.

Logical Operators in Python

Logical operators allow you to combine multiple conditions in your code, enabling you to perform complex checks based on several criteria. Python has three primary logical operators:

  • AND
  • OR
  • NOT

AND Operator (and)

The and operator checks if both conditions are true. If either condition is false, the entire expression will evaluate to false. 

age = 25
if age >= 18 and age <= 35:
    print("You are a young adult.")

Explanation: The condition age >= 18 and age <= 35 will only be true if age is both 18 or older and 35 or younger. If age were 40, this statement would not print anything because age <= 35 would be false.


More Examples with AND

salary = 5000
years_experience = 3
if salary > 3000 and years_experience > 2:
    print("Eligible for senior position.")

This example checks that the person has a salary greater than 3000 and more than 2 years of experience.

OR Operator (or)

The or operator checks if at least one condition is true. If any one of the conditions is true, the entire expression will evaluate to true. 

temperature = 30
if temperature < 15 or temperature > 25:
    print("The weather is outside the comfortable range.")

Explanation: This code checks if the temperature is below 15 or above 25. If either condition is true, the message will print.


More Examples with OR

day = "Saturday"
if day == "Saturday" or day == "Sunday":
    print("It's the weekend!")

This example checks if the day is either Saturday or Sunday to determine if it's the weekend.

NOT Operator (not)

The not operator reverses the Boolean value of a condition. If the condition is true, not makes it false, and if it’s false, not makes it true. 

is_raining = False
if not is_raining:
    print("You can leave the umbrella at home.")

Explanation: not is_raining inverts the value of is_raining. So if is_raining is false, not is_raining is true, and the message will print.


More Examples with NOT

        is_sunny = True
        if not is_sunny:
            print("It might be cloudy or rainy today.")

This example checks if is_sunny is false (i.e., it’s not sunny) and prints a message accordingly.

Summary Table

Operator Description Example Result
and True if both conditions are true True and False False
or True if at least one condition is true True or False True
not Reverses the Boolean value of a single condition not True False
