Python Practice Exercises - Tuples, Sets, and Dictionaries

Complete the following exercises to strengthen your understanding of Tuples, Sets, and Dictionaries concepts.

1. Find the Most Frequent Element in a List

Write a program that finds the most frequent element in a list and returns it. 

my_list = [1, 3, 2, 3, 4, 5, 3, 2, 1]
Solution 
my_list = [1, 3, 2, 3, 4, 5, 3, 2, 1]
freq_dict = {}
for item in my_list:
    freq_dict[item] = freq_dict.get(item, 0) + 1
most_frequent = max(freq_dict, key=freq_dict.get)
print(most_frequent)

my_list = ['apple', 'banana', 'orange', 'apple', 'banana', 'apple']

3. Find the Most Frequent Palindromic String

Write a program that finds the most frequent palindromic string from a list. A palindrome is a string that reads the same forward and backward. 

my_list = ['level', 'radar', 'apple', 'madam', 'level', 'radar']
Solution 
my_list = ['level', 'radar', 'apple', 'madam', 'level', 'radar']
palindromes = [word for word in my_list if word == word[::-1]]
freq_dict = {}
for word in palindromes:
    freq_dict[word] = freq_dict.get(word, 0) + 1
most_frequent_palindrome = max(freq_dict, key=freq_dict.get)
print(most_frequent_palindrome)

4. Count of Unique Elements in List

Write a program that counts how many unique elements are in a list.

my_list = [1, 2, 3, 3, 4, 5, 5]
Solution 
my_list = [1, 2, 3, 3, 4, 5, 5]
unique_count = len(set(my_list))
print(unique_count)

6. Find the Frequency of Words in a Sentence

Write a program that counts the frequency of each word in a sentence and returns the result as a dictionary. 

sentence = "this is a test this is only a test"
Solution 
sentence = "this is a test this is only a test"
words = sentence.split()
freq_dict = {}
for word in words:
    freq_dict[word] = freq_dict.get(word, 0) + 1
print(freq_dict)

8. Count Palindromic Words in a List

Write a program that counts how many palindromic words (words that read the same backward) are in a list. 

my_list = ['madam', 'apple', 'level', 'banana', 'radar']
Solution 
my_list = ['madam', 'apple', 'level', 'banana', 'radar']
palindromes = [word for word in my_list if word == word[::-1]]
print(len(palindromes))

9. Remove Elements with Frequency Less Than a Certain Threshold

Write a program that removes all elements from a list that appear less than a specified threshold frequency. 

my_list = [1, 2, 2, 3, 4, 5, 2, 6, 2]
Solution 
my_list = [1, 2, 2, 3, 4, 5, 2, 6, 2]
threshold = 3
freq_dict = {}
for item in my_list:
    freq_dict[item] = freq_dict.get(item, 0) + 1
filtered_list = [item for item in my_list if freq_dict[item] >= threshold]
print(filtered_list)

10. Find Unique Even Numbers in a List

Write a program that finds all the unique even numbers in a list.

my_list = [1, 2, 3, 2, 4, 5, 6, 3, 4]
Solution 
my_list = [1, 2, 3, 2, 4, 5, 6, 3, 4]
even_numbers = {num for num in my_list if num % 2 == 0}
print(even_numbers)

11. Find Unique Strings of a Specific Length

Write a program that finds all the unique strings of length 3 or more from a list of strings. 

my_list = ['apple', 'banana', 'pear', 'grape', 'apple', 'grape', 'kiwi']
Solution 
my_list = ['apple', 'banana', 'pear', 'grape', 'apple', 'grape', 'kiwi']
long_strings = {word for word in my_list if len(word) >= 3}
print(long_strings)
