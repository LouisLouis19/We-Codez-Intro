Python Practice Exercises

Complete the following exercises to strengthen your understanding of If-Else Concepts

1. Simple If/Else Statement

Check if a number is positive or negative:

number = -5
Solution 
number = -5
if number >= 0:
    print("The number is positive.")
else:
    print("The number is negative.")

2. Elif Statement

Determine the life stage of a person based on their age.

  • Child: age is less than 13
  • Teenager: age is between 13 and 19, inclusive
  • Adult: age is between 20 and 64, inclusive
  • Senior: age is 65 or older
age = 65
Solution 
age = 65
if age < 13:
    print("Child")
elif 13 <= age < 20:
    print("Teenager")
elif 20 <= age < 65:
    print("Adult")
else:
    print("Senior")

3. Check Even or Odd

Write a program that checks if a number is even or odd:

number = 7
Solution 
number = 7
if number % 2 == 0:
    print("The number is even.")
else:
    print("The number is odd.")

4. Check Divisibility by 3, 5, or Both

Create a program that checks if a number is divisible by 3, 5, or both: 

number = 15
Solution 
number = 15
if number % 3 == 0 and number % 5 == 0:
    print("Divisible by both 3 and 5.")
elif number % 3 == 0:
    print("Divisible by 3.")
elif number % 5 == 0:
    print("Divisible by 5.")
else:
    print("Not divisible by 3 or 5.")

5. Determine the Largest of Three Numbers

Write a program to find the largest of three numbers:

a = 5, b = 12, c = 7
Solution 
a, b, c = 5, 12, 7
if a >= b and a >= c:
    print("The largest number is:", a)
elif b >= a and b >= c:
    print("The largest number is:", b)
else:
    print("The largest number is:", c)

6. Check Leap Year

Write a program to check if a year is a leap year. A year is a leap year if:

  • It is divisible by 4, and not divisible by 100, or
  • It is divisible by 400.

Example:

year = 2020
Solution 
year = 2020
if (year % 4 == 0 and year % 100 != 0) or (year % 400 == 0):
    print("Leap year")
else:
    print("Not a leap year")

7. Check Temperature Range

Write a program that categorizes the temperature based on its value.

  • If the temperature is 30 degrees Celsius or above, print "It's hot."
  • If the temperature is between 20 (inclusive) and 30 (exclusive) degrees Celsius, print "It's warm."
  • If the temperature is between 10 (inclusive) and 20 (exclusive) degrees Celsius, print "It's cool."
  • If the temperature is below 10 degrees Celsius, print "It's cold." 
temperature = 30
Solution 
temperature = 30
if temperature >= 30:
    print("It's hot.")
elif 20 <= temperature < 30:
    print("It's warm.")
elif 10 <= temperature < 20:
    print("It's cool.")
else:
    print("It's cold.")

8. Determine Zero, Odd or Even

Write a program that determines if a given number is zero, odd, or even. Print the result. 

number = 3
Solution 
number = 3
if number == 0:
    print("The number is zero")
elif number % 2 == 0:
    print("The number is even")
else:
    print("The number is odd")

9. Grade Classification

Write a program that reads a score from the user and classifies it into grades: 'A' for scores 90 and above, 'B' for scores between 80 and 89, 'C' for scores between 70 and 79, and 'F' for scores below 70. Print the grade. 

score = 85
Solution 
score = 85
if score >= 90:
    print("Grade: A")
elif score >= 80:
    print("Grade: B")
elif score >= 70:
    print("Grade: C")
else:
    print("Grade: F")
