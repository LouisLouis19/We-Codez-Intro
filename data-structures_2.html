Introduction to Data Structures

Python provides several built-in data structures to store and manage data efficiently. This section introduces tuples, sets, and dictionaries, which are versatile and powerful tools for handling data.

Python Tuples

Tuples are immutable sequences used to store a collection of items. They are similar to lists, but unlike lists, tuples cannot be changed (i.e., immutable). Tuples are defined using parentheses () and can hold elements of different data types. They are often used when the data should not be modified after creation.

Creating Tuples

Here are some examples of creating tuples:

# Creating tuples:
empty_tuple = ()
single_element_tuple = (42,)  # Note the comma for single-element tuples
mixed_tuple = (1, "apple", 3.14, True)
nested_tuple = (1, (2, 3), [4, 5])

print(mixed_tuple)   # Output: (1, "apple", 3.14, True)
print(nested_tuple)  # Output: (1, (2, 3), [4, 5])

Accessing Tuple Elements

You can access tuple elements using zero-based indexing or negative indexing (to count from the end). Tuples also support slicing to retrieve portions of the tuple. 

my_tuple = (10, 20, 30, 40, 50)

# Accessing elements:
print(my_tuple[0])    # Output: 10 (first element)
print(my_tuple[-1])   # Output: 50 (last element)

# Slicing tuples:
print(my_tuple[1:4])  # Output: (20, 30, 40)
print(my_tuple[:3])   # Output: (10, 20, 30)
print(my_tuple[::2])  # Output: (10, 30, 50) (every second element)

Tuple Unpacking

Tuple unpacking allows you to assign the values of a tuple to multiple variables in one step. The number of variables must match the number of elements in the tuple. 

# Example of tuple unpacking:
my_tuple = (1, 2, 3)
a, b, c = my_tuple

print(a)  # Output: 1
print(b)  # Output: 2
print(c)  # Output: 3

# Unpacking with the asterisk (*) operator:
numbers = (1, 2, 3, 4, 5)
first, *middle, last = numbers
print(first)   # Output: 1
print(middle)  # Output: [2, 3, 4]
print(last)    # Output: 5

Looping Through Tuples

You can iterate over the elements of a tuple using a for loop: 

# Example of looping through a tuple:
fruits = ("apple", "banana", "cherry")
for fruit in fruits:
    print(fruit)

# Output:
# apple
# banana
# cherry

Joining Tuples

Tuples can be concatenated using the + operator and repeated using the * operator: 

tuple1 = (1, 2, 3)
tuple2 = (4, 5, 6)

# Concatenation:
result = tuple1 + tuple2
print(result)  # Output: (1, 2, 3, 4, 5, 6)

# Repetition:
repeated = tuple1 * 2
print(repeated)  # Output: (1, 2, 3, 1, 2, 3)

Tuple Methods

Tuples have two main built-in methods: count() and index().

  • count(): Returns the number of occurrences of a specified value in the tuple.
  • index(): Returns the index of the first occurrence of a specified value. Raises a ValueError if the value is not found. 
# Using tuple methods:
numbers = (10, 20, 10, 30, 40, 10)

# Count the occurrences of 10:
print(numbers.count(10))  # Output: 3

# Find the index of the first occurrence of 30:
print(numbers.index(30))  # Output: 3

Immutability of Tuples

Tuples are immutable, meaning their content cannot be changed after creation. If you need a similar data structure that allows modification, consider using a list. 

# Trying to modify a tuple will raise an error:
my_tuple = (1, 2, 3)
# my_tuple[1] = 20  # This will raise a TypeError

# However, if a tuple contains mutable elements (e.g., a list), those elements can be modified:
nested_tuple = (1, [2, 3], 4)
nested_tuple[1][0] = 99
print(nested_tuple)  # Output: (1, [99, 3], 4)

Python Sets

Sets are unordered collections of unique elements. They are defined using curly braces {} or the set() function. Since sets do not allow duplicate elements, they are useful for tasks that require unique data.

Creating Sets

Here are some examples of creating sets:

# Examples of sets:
fruits = {"apple", "banana", "cherry"}
empty_set = set()  # Note: {} creates an empty dictionary, not a set
numbers = set([1, 2, 3, 4, 4])  # Duplicate values are removed
print(numbers)  # Output: {1, 2, 3, 4}

Accessing Set Elements

Sets are unordered, so they do not support indexing or slicing. However, you can iterate through the elements of a set using a loop: 

# Example of iterating through a set:
fruits = {"apple", "banana", "cherry"}
for fruit in fruits:
    print(fruit)

# Output (order may vary):
# apple
# banana
# cherry

Adding Elements to a Set

You can add elements to a set using the add() method for single elements or the update() method for multiple elements. 

# Adding elements to a set:
fruits = {"apple", "banana"}
fruits.add("cherry")
print(fruits)  # Output: {"apple", "banana", "cherry"}

# Adding multiple elements:
fruits.update(["orange", "grape"])
print(fruits)  # Output: {"apple", "banana", "cherry", "orange", "grape"}

Removing Elements from a Set

You can remove elements using the remove() or discard() methods. The remove() method raises an error if the element is not found, while discard() does not. 

# Removing elements:
fruits = {"apple", "banana", "cherry"}
fruits.remove("banana")
print(fruits)  # Output: {"apple", "cherry"}

# Using discard (no error if element doesn't exist):
fruits.discard("grape")
print(fruits)  # Output: {"apple", "cherry"}

# Remove and return a random element:
popped_element = fruits.pop()
print(popped_element)  # Random element from set
print(fruits)  # Remaining elements

Python Dictionaries

Dictionaries store data as key-value pairs, making them ideal for associating related information. They are defined using curly braces {}, and keys must be unique and immutable (e.g., strings, numbers, or tuples). Values can be of any data type.

Creating Dictionaries

Here are some examples of creating dictionaries:

# Example of a dictionary:
student = {"name": "Alice", "age": 25, "course": "Python"}

# Creating an empty dictionary:
empty_dict = {}

# Using the dict() function:
person = dict(name="Bob", age=30, profession="Engineer")
print(person)  # Output: {'name': 'Bob', 'age': 30, 'profession': 'Engineer'}

Accessing and Modifying Values

You can access a dictionary value using its key. If the key does not exist, a KeyError is raised unless you use the get() method. 

# Accessing values:
student = {"name": "Alice", "age": 25, "course": "Python"}
print(student["name"])  # Output: Alice

# Using the get() method (returns None if the key is missing):
print(student.get("grade"))  # Output: None
print(student.get("grade", "Not Assigned"))  # Output: Not Assigned

Adding and Modifying Key-Value Pairs

Adding a new key-value pair is as simple as assigning a value to a new key. You can also modify an existing value by reassigning it to the key. 

# Adding a new key-value pair:
student["grade"] = "A"
print(student)  # Output: {'name': 'Alice', 'age': 25, 'course': 'Python', 'grade': 'A'}

# Modifying an existing value:
student["age"] = 26
print(student)  # Output: {'name': 'Alice', 'age': 26, 'course': 'Python', 'grade': 'A'}

Removing Key-Value Pairs

You can remove key-value pairs using methods like pop(), popitem(), and del. 

# Removing a specific key:
grade = student.pop("grade")  # Removes and returns the value
print(grade)  # Output: A
print(student)  # Output: {'name': 'Alice', 'age': 26, 'course': 'Python'}

# Removing the last key-value pair (Python 3.7+):
last_item = student.popitem()
print(last_item)  # Output: ('course', 'Python')
print(student)  # Output: {'name': 'Alice', 'age': 26}

# Using del:
del student["age"]
print(student)  # Output: {'name': 'Alice'}

Looping Through Dictionaries

You can loop through dictionaries to access keys, values, or both. 

# Looping through keys:
for key in student:
    print(key)

# Looping through values:
for value in student.values():
    print(value)

# Looping through key-value pairs:
for key, value in student.items():
    print(f"{key}: {value}")

# Output:
# name: Alice

Common Dictionary Methods

Python provides several useful methods for working with dictionaries:

  • keys(): Returns a view object containing all keys.
  • values(): Returns a view object containing all values.
  • items(): Returns a view object containing all key-value pairs as tuples.
  • update(): Merges another dictionary into the current dictionary.
  • clear(): Removes all key-value pairs.
  • copy(): Returns a shallow copy of the dictionary.
# Using common dictionary methods:
student = {"name": "Alice", "age": 25, "course": "Python"}

# Getting keys, values, and items:
print(student.keys())    # Output: dict_keys(['name', 'age', 'course'])
print(student.values())  # Output: dict_values(['Alice', 25, 'Python'])
print(student.items())   # Output: dict_items([('name', 'Alice'), ('age', 25), ('course', 'Python')])

# Merging dictionaries:
additional_info = {"grade": "A", "city": "New York"}
student.update(additional_info)
print(student)  # Output: {'name': 'Alice', 'age': 25, 'course': 'Python', 'grade': 'A', 'city': 'New York'}

# Clearing a dictionary:
student.clear()
print(student)  # Output: {}

1. Tuple Indexing

Write a program to access the last element of a tuple using negative indexing.

Solution 
my_tuple = (5, 10, 15, 20)
print(my_tuple[-1])

2. Tuple Unpacking

Given a tuple with three elements, unpack the values into three variables and print them.

Solution 
my_tuple = (1, 2, 3)
a, b, c = my_tuple
print(a, b, c)

Write a program to print each element of a tuple on a new line.

Solution 
my_tuple = (100, 200, 300)
for item in my_tuple:
    print(item)

Count the occurrences of the number 3 in the tuple my_tuple = (1, 3, 3, 7, 3, 9).

Solution 
my_tuple = (1, 3, 3, 7, 3, 9)
print(my_tuple.count(3))

Check if the number 5 is present in the set numbers = {1, 3, 5, 7}.

Solution 
numbers = {1, 3, 5, 7}
print(5 in numbers)

Add the element 10 to the set my_set = {1, 2, 3}.

Solution 
my_set = {1, 2, 3}
my_set.add(10)
print(my_set)

Write a program to print each element of the set fruits = {"apple", "banana", "cherry"}.

Solution 
fruits = {"apple", "banana", "cherry"}
for fruit in fruits:
    print(fruit)

8. Dictionary Lookup

Retrieve the value associated with the key "course" from the dictionary student = {"name": "Bob", "course": "Math", "age": 20}.

Solution 
student = {"name": "Bob", "course": "Math", "age": 20}
print(student["course"])

9. Adding a Key-Value Pair

Add a new key-value pair "grade": "A" to the dictionary student = {"name": "Alice", "age": 25}.

Solution 
student = {"name": "Alice", "age": 25}
student["grade"] = "A"
print(student)

10. Looping Through a Dictionary

Print all keys and values from the dictionary person = {"name": "Sam", "age": 29, "city": "Chicago"}.

Solution 
person = {"name": "Sam", "age": 29, "city": "Chicago"}
for key, value in person.items():
    print(f"{key}: {value}")

11. Removing a Key

Remove the key "age" from the dictionary person = {"name": "Sam", "age": 29, "city": "Chicago"}.

Solution 
person = {"name": "Sam", "age": 29, "city": "Chicago"}
del person["age"]
print(person)

12. Dictionary Default Value

Use the get() method to safely access the value for the key "grade" in the dictionary student = {"name": "Alice", "age": 25}. If the key does not exist, return "Not Assigned".

Solution 
student = {"name": "Alice", "age": 25}
print(student.get("grade", "Not Assigned"))
