Overview

This week, we will explore the basics of functions in Python. Functions are a fundamental concept in programming that allow you to organize your code, make it reusable, and improve its readability. Understanding how to create and use functions is essential for writing clean, maintainable code.

Functions help you break down complex problems into smaller, manageable pieces. Additionally, functions can be reused across different parts of your program, saving you time and effort in coding.

What Are Functions?

A function is a block of organized, reusable code that performs a specific task. Functions provide better modularity for your application and a high degree of code reusability.

Using functions effectively can lead to cleaner code that is easier to read and understand. It also promotes code reuse, meaning you can write a function once and use it multiple times throughout your program. This reduces redundancy and helps maintain consistency in your code.

Functions typically take input parameters and may return a value. This allows you to customize their behavior based on the inputs you provide. For example, a function that adds two numbers together can be called with different pairs of numbers to get different results.

Creating a Function in Python

In Python, a function is defined using the def keyword, followed by the function name and parentheses (). Functions allow you to encapsulate code into reusable blocks, making your programs more organized and easier to maintain.

Basic syntax:

def function_name():
    # Code block to be executed when the function is called
    pass

Example of a simple function:

def greet():
    print("Hello, welcome to Python functions!")

To call a function, use the function name followed by parentheses: 

greet()  # Output: Hello, welcome to Python functions!

Functions can help reduce code duplication and enhance readability. For example, if you need to greet multiple users, you can simply call the greet function multiple times instead of rewriting the greeting message each time.

Function Parameters

Functions can accept parameters, which allow you to pass data into the function. These parameters are specified within the parentheses when defining the function. Parameters help you create more flexible and reusable functions.

Example of a function with a single parameter:

def greet(name):
    print("Hello, " + name + "! Welcome to Python functions!")

You can now pass an argument when calling the function:

greet("John")  # Output: Hello, John! Welcome to Python functions!
greet("Sam")  # Output: Hello, Sam! Welcome to Python functions!

You can also define functions with multiple parameters to perform operations on different values. For instance: 

def add_numbers(a, b):
    print(a + b)

To call this function and obtain the result:

add_numbers(5, 3) # Output: 8

The Return Statement

The return statement is used to send a result back to the caller. If a function doesn't have a return statement, it returns None by default. This makes the function versatile, allowing it to compute values and return results when needed.

Example of a function that returns a value:

def multiply(a, b):
    return a * b

When you call the multiply function, it calculates the product of a and b and sends the result back to the caller: 

result = multiply(4, 5)
print(result)  # Output: 20

The result can then be stored in a variable, used in further calculations, or printed directly. For instance, you can use the returned value in another calculation: 

total = result + 10
print(total)  # Output: 30

Practice Problems

1. Simple Greeting Function

Create a function called `greet_user` that takes a name as a parameter and prints "Hello, [name]!".

Solution 
def greet_user(name):
    print("Hello, " + name + "!")

greet_user("Alice")  # Output: Hello, Alice!

2. Temperature Converter

Create a function called `celsius_to_fahrenheit` that takes a temperature in Celsius as a parameter and returns the temperature in Fahrenheit.

Solution 
def celsius_to_fahrenheit(celsius):
    return (celsius * 9/5) + 32

print(celsius_to_fahrenheit(25))  # Output: 77.0

3. Calculate Area of a Circle

Create a function called `area_of_circle` that takes the radius as a parameter and returns the area of the circle. (Use the formula: `area = π * r^2`, where `π` can be approximated as `3.14159`.)

Solution 
def area_of_circle(radius):
    pi = 3.14159
    return pi * (radius ** 2)

print(area_of_circle(5))  # Output: 78.53975

4. Check Even or Odd

Create a function called `is_even` that takes an integer as a parameter and returns `True` if the number is even, and `False` if it's odd.

Solution 
def is_even(number):
    return number % 2 == 0

print(is_even(10))  # Output: True
print(is_even(7))   # Output: False

5. Find the Maximum of Two Numbers

Create a function called `max_of_two` that takes two numbers as parameters and returns the larger of the two.

Solution 
def max_of_two(a, b):
    if a > b:
        return a
    else:
        return b

print(max_of_two(10, 20))  # Output: 20
